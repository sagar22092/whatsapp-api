WhatsApp API Documentation

A comprehensive API for sending messages, media, locations, polls, and managing WhatsApp interactions programmatically.

Base URL

All API endpoints start with the following base URL:

http://localhost:3002

All API endpoints are prefixed with /api/whatsapp after this base URL.

Text Photo Video Audio File Location Poll Reaction Delete Groups

Introduction

This API allows you to interact with WhatsApp programmatically to send messages, media files, locations, polls, reactions, and perform other messaging operations.

All endpoints require authentication via an API key passed in the request headers. The base URL for all endpoints is http://localhost:3002 with the API prefix /api/whatsapp.

Authentication

All API requests must include an X-API-Key header with a valid API key associated with an active WhatsApp session.

// Example Header X-API-Key: "your-api-key-here"

Without a valid API key, requests will return a 400 or 404 error.

POST http://localhost:3002/api/whatsapp/send/text

Send a text message to a phone number or WhatsApp group.

Request Headers

Header Type Required Description
X-API-Key String Yes Your API key for authentication

Request Body (JSON)

Parameter Type Required Description
number String Optional* Recipient phone number (with country code, without '+' or spaces)
group String Optional* Group ID for sending to a WhatsApp group
message String Yes Text message content

* Either number or group must be provided

Example Request

// Using curl curl -X POST \ -H "X-API-Key: your-api-key-here" \ -H "Content-Type: application/json" \ -d '{ "number": "1234567890", "message": "Hello from WhatsApp API!" }' \ http://localhost:3002/api/whatsapp/send/text
POST http://localhost:3002/api/whatsapp/send/photo

Send one or multiple images (up to 10) to a phone number or WhatsApp group.

Request Format

This endpoint uses multipart/form-data for file uploads.

Parameters

Parameter Type Required Description
photos File[] Yes Image files (max 10)
number String Optional* Recipient phone number
group String Optional* Group ID for sending to a group
caption String No Caption for the image(s)
viewOnce Boolean No If true, sends as view-once media

Example Request

// Using JavaScript fetch const formData = new FormData(); formData.append('photos', photoFile); formData.append('number', '1234567890'); formData.append('caption', 'Check out this image!'); const response = await fetch('http://localhost:3002/api/whatsapp/send/photo', { method: 'POST', headers: { 'X-API-Key': 'your-api-key-here' }, body: formData });

POST http://localhost:3002/api/whatsapp/send/location

Send a location with latitude and longitude coordinates to a phone number or WhatsApp group.

Request Body (JSON)

Parameter Type Required Description
number String Optional* Recipient phone number
group String Optional* Group ID for sending to a group
location.degreesLatitude Number Yes Latitude coordinate (e.g., 40.7128)
location.degreesLongitude Number Yes Longitude coordinate (e.g., -74.0060)

* Either number or group must be provided

Example Request

// Sending location coordinates for New York City const response = await fetch('http://localhost:3002/api/whatsapp/send/location', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': 'your-api-key-here' }, body: JSON.stringify({ number: '1234567890', location: { degreesLatitude: 40.7128, degreesLongitude: -74.0060 } }) });
POST http://localhost:3002/api/whatsapp/send/poll

Create and send a poll to a phone number or WhatsApp group.

Parameters

Parameter Type Required Description
number String Optional* Recipient phone number
group String Optional* Group ID for sending to a group
name String Yes Poll question/title
values String[] Yes Array of poll options (minimum 2)
selectableCount Number No Number of options users can select (default: 1)
toAnnouncementGroup Boolean No If sending to an announcement group
POST http://localhost:3002/api/whatsapp/message/react

React to a message with an emoji.

Parameters

Parameter Type Required Description
number String Optional* Recipient phone number
group String Optional* Group ID for sending to a group
messageKey Object Yes Message key object containing message details
emoji String Yes Emoji character for the reaction

Example Request

// React with 👍 to a message const response = await fetch('http://localhost:3002/api/whatsapp/message/react', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': 'your-api-key-here' }, body: JSON.stringify({ number: '1234567890', messageKey: { remoteJid: '1234567890@s.whatsapp.net', fromMe: true, id: '3EB0A5EF77EAFADBEFF2D6' }, emoji: '👍' }) });
POST http://localhost:3002/api/whatsapp/message/delete

Delete a message for everyone in the chat.

Parameters

Parameter Type Required Description
number String Optional* Recipient phone number
group String Optional* Group ID for sending to a group
messageKey Object Yes Message key object containing message details
GET http://localhost:3002/api/whatsapp/get/groups

Retrieve a list of all WhatsApp groups for the authenticated session.

Request Headers

Header Type Required Description
X-API-Key String Yes Your API key for authentication

Example Request

// Get all WhatsApp groups const response = await fetch('http://localhost:3002/api/whatsapp/get/groups', { method: 'GET', headers: { 'X-API-Key': 'your-api-key-here' } }); const data = await response.json(); console.log(data.groups);
